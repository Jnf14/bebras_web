Body

Ein Biber möchte einen Damm bauen. Damit er immer die richtigen Bäume fällt, hat er sich zwei Bedingungen überlegt. Er wird einen Baum nur fällen, wenn beide erfüllt sind:

  • Direkt links daneben befindet sich ein kleinerer Baum.

  • Direkt rechts daneben befindet sich ein größerer Baum.

Question / Challenge

Klicke auf die Bäume, die der Biber fällen wird!

Answer Options / Interactivity Description

The student should be able to click(toggle) every single tree. The selection should be saved while switching tasks. When finished the student has to save(commit) to the result.

Answer Explanation

Nur die Bäume an vierter Stelle und siebter Stelle erfüllen die vorgegebenen Bedingungen: Auf der linken Seite befindet sich ein kleinerer Baum UND auf der rechten Seite ein größerer Baum.

It's Informatics!

In der Informatik geht es oft darum, Probleme zu lösen, die durch eine Reihe von logischen Einschränkungen (engl. Constraints) spezifiziert sind. Die Aufgabe besteht darin, eine Lösung zu finden, die alle gegebenen Einschränkungen erfüllt. Komplexere Aufgaben als diese können behandelt werden, indem die Einschränkungen mit Hilfe von logische Operatoren kombiniert werden. Die Konjunktion (∧ oder auch UND-Operator) beispielsweise liefert im Ausdruck A ∧ B genau dann Wahr als Ergebnis, wenn die beiden Einschränkungen ebenfalls wahr sind. In dieser Aufgabe wäre das dann also: (der Baum links ist kleiner) ∧ (der Baum rechts ist grösser). Dieses grundlegende Prinzip kommt in nahe zu allen Bereichen der Informatik vor, wie beispielsweise bei vielen Sortier-Algorithmen, zum Beispiel dem Bubble Sort, hier werden immer mehrere Objekte einer Liste, auf gewisse Einschränkungen überprüft, um Sie danach, sofern notwendig, an eine andere Position in der Liste zu verschieben, dieses Prinzip wird, solange wiederholt bis die Liste vollständig sortiert ist.
